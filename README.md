# Magic Prompts

A premium library of curated AI image prompts for ChatGPT Images, Gemini, Midjourney, Flux, Leonardo AI, Ideogram, Stable Diffusion, Adobe Firefly, and more.

## Technologies Used

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **State & Data**: TanStack React Query (v5)
- **UI Components**: Radix UI (accessible primitives) & Lucide Icons
- **Animations**: Framer Motion

## Development Setup

To run this application locally, follow these steps:

1. **Install dependencies**:

   ```bash
   bun install
   # or
   npm install
   ```

2. **Start the development server**:

   ```bash
   bun run dev
   # or
   npm run dev
   ```

3. **Build for production**:
   ```bash
   bun run build
   # or
   npm run build
   ```
